Gathering of Personally-Identifying Information

BarRemote allows you to control Spotify playback and see what's playing in the Spotify app. It does this by communicating with the Spotify app installed locally on your machine. It also offers a "search" feature where users can search for an artist name, album name, or track name, and the results are displayed inside the app.

BarRemote does not collect or store any personal data. Communication with the Spotify app is done by sending commands such as "play next track", "play previous track", "pause playback", "start playback", "set shuffle on", etc.
The information used for the search feature is send to the Spotify Web API, in order to provide the results of the search. No user identifiable information is send to the Spotify service with these requests. Only the search text is send to the Spotify APIs.
